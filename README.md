# How to replicate this project
## protobuf
Extract .tar.gz file:
```
tar -xzf protobuf-cpp-x.x.x.tar.gz
```
cd into the extracted directory:
```
cd protobuf-x.x.x
```
Build the code:
```
./configure
make
make check
sudo make install
```
Verify that the installation was successful:
```
protoc --version
```
You may need to load it up!
```
sudo ldconfig
protoc --version
```
